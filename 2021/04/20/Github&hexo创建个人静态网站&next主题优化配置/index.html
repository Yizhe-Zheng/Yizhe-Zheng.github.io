Hexo 命令介绍

常用的Hexo命令

如果配置了博客文件，Ctrl+F5强制刷新后页面不见有变化，可以使用命令hexo clean清理已生成博客网页文件，之后使用hexo g命令重新生成。

设置网站图标

在EasyIcon中分别找一张16*16与32*32的ico图标,或者去别的网站下载或者自己制作，并将图标名称改为favicon16.ico与favicon32.ico，然后把图标放在/themes/next/source/images或者放在根目录的/source/images文件夹里。
在主题配置文件内搜索favicon字段，把small、medium字段的地址修改为/images/favicon16.ico与/images/favicon32.ico即可。

设置赞赏功能

主题自带支付宝跟微信及比特币打赏功能，只需要主题配置文件中配置启用，并设置微信和支付宝收款二维码图片地址即可开启功能。搜索reward字段，将wechatpay及alipay字段前面的#去掉即可。在source文件夹中放置支付宝跟微信收款码并修改文件名字即可。comment字段自带为打赏按钮上方的文字，不需要的话可以在前面加#注释掉。

图床

当博文中有少量图片是，可以直接把图片存放在source文件夹中，但图片会占据大量的存储的空间，加载相对缓慢，我们可以考虑把文章里的图片上传到某个网站，然后获得外部链接，写文章时使用Markdown语法，![图片信息](外部链接), 完成图片的插入，这种网站就被称为图床。建议使用比较知名的，比如：腾讯云、网易蜂巢、七牛云、京东云等。
